# Day-2 Deliverables Mapping

This file maps implementation artifacts to the required Day-2 problem statement.

## 1. Labeled Dataset (Resume + JD + Fit Score 0-10)

- Generator script:
  - `ml_pipeline/scripts/generate_synthetic_dataset.py`
- Outputs:
  - `ml_pipeline/data/train.jsonl`
  - `ml_pipeline/data/val.jsonl`
  - `ml_pipeline/data/test.jsonl`
  - `ml_pipeline/data/dataset_meta.json`

Each sample includes:
- `resume_text`
- `job_description`
- `fit_score` (0-10)
- `skills`
- `skill_spans`
- `overclaim_flags`

Production mode:
- replace generated files with real labeled `train/val/test.jsonl`
- keep same schema fields for training/evaluation scripts

## 2. Fine-Tuned Transformer Models

Score model (resume-to-job similarity / scoring, ordinal 0-10):
- `ml_pipeline/scripts/train_score_model.py`
- Artifact: `ml_pipeline/artifacts/score_model/`

Skill extraction model:
- `ml_pipeline/scripts/train_skill_model.py`
- Artifact: `ml_pipeline/artifacts/skill_model/`

Candidate ranking:
- derived by sorting predicted fit scores per `job_id`

## 3. Baseline Comparisons

Implemented in:
- `ml_pipeline/scripts/evaluate_models.py`

Baselines:
- Embedding similarity baseline (`sentence-transformers/all-MiniLM-L6-v2`)
- Zero-shot LLM baseline (`google/flan-t5-small`)

## 4. Evaluation Metrics

Generated by:
- `ml_pipeline/scripts/evaluate_models.py`

Reports:
- `ml_pipeline/reports/evaluation_report.json`
- `ml_pipeline/reports/evaluation_report.md`

Included metrics:
- Skill extraction: Precision, Recall, F1, Exact-match Accuracy
- Scoring/ranking: MAE, Spearman ranking correlation, NDCG@10

## 5. Bias Analysis

Implemented in:
- `ml_pipeline/scripts/run_bias_audit.py`

Report:
- `ml_pipeline/reports/bias_audit_report.json`

Includes:
- Counterfactual demographic swap score deltas
- Flagged ratio above configured threshold
- Demographic parity difference

## 6. Lightweight Inference Demo

Implemented in:
- `ml_pipeline/scripts/demo_inference.py`

Output:
- `ml_pipeline/reports/demo_inference.json`

## 7. Full End-to-End Orchestration

Run all steps:
- `ml_pipeline/scripts/run_day2_pipeline.py`
- `scripts/deep_audit.ps1`

Production-strength defaults validated on 16GB RAM:
- `NumJobs=120`
- `CandidatesPerJob=8`
